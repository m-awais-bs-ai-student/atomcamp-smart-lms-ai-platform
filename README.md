# ⚛️ Atomcamp Smart LMS – AI Powered Adaptive Learning Platform

## 🧠 Project Description

Atomcamp Smart LMS is an AI-powered Learning Management System (LMS) built with **Streamlit** and **Google Gemini API**.  
It provides a fully adaptive learning experience including AI tutoring, smart quizzes, personalized learning paths, and analytics dashboards for learners and instructors.

This project demonstrates how Generative AI can be used to build intelligent, personalized education systems.

---

## 🚀 Features

### 🎓 Learner Features
- Personalized onboarding system  
- Course enrollment system  
- Progress tracking with XP system  
- Learning streaks and gamification  

### 🤖 AI Tutor
- Powered by Google Gemini AI  
- Answers academic and technical questions  
- Provides explanations and learning guidance  
- Context-aware responses using learner profile  

### 📝 Smart Quiz System
- AI-generated MCQs  
- Difficulty levels (Beginner → Advanced)  
- Instant evaluation with explanations  
- XP rewards for correct answers  

### 📊 Analytics Dashboard
- Course progress tracking  
- Activity visualization  
- Skill radar chart  
- Learning behavior insights  

### 👨‍🏫 Instructor Dashboard
- Cohort performance monitoring  
- At-risk learner detection  
- Engagement analytics  
- AI-based curriculum insights
